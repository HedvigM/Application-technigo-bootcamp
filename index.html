CSS

Everything you need to know

CSS is a style sheet language that is complementary to a markup language (such as HTML). A markup language is used to describe the content of a web page and CSS to describe how the page should be presented (layout, colors, fonts, etc.). The appearance is built using English keywords. When CSS was launched in 1996, it was revolutionary for the industry. The division between the languages has led to many advantages, not least to increase the readability of the code and making it easier to maintain over time.
